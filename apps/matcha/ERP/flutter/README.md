# Matcha ERP — Flutter

Flutter web wrapper for self-hosted ERPNext instances.

**Scaffold from:** [flutter-template/](../../../../flutter-template/)

Follow [flutter-template/docs/FORKING.md](../../../../flutter-template/docs/FORKING.md), then set in `lib/core/config/app_config.dart`:

```dart
static const String appName = 'Matcha ERP';
static const String appId = 'com.matcha.erp';
```

This directory will hold the forked Flutter project once initialized.
